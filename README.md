# Gotcha mod
    Collection of tools for Pokemon Go (in Developing)
# Setup
- Install Nodejs and related package ('discord.js')
- Update config.json before run.
```json
{ 
    "token"  : "",
    "prefix" : "."
}
```
- Run command
```
    node app.js
```

# Usage
 $ Check iv of Raid/Egg/Quest pokemon:
```
    iv <pokemon> <cp>
```
